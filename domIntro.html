Understanding the DOM in JavaScript

What is the DOM?

DOM stands for Document Object Model. It is a programming interface that represents an HTML document as a tree of objects. JavaScript uses the DOM to access, change, add, or remove elements on a web page.

When a browser loads an HTML page, it converts the HTML into the DOM.

DOM Tree Structure

Every HTML element becomes a node in the DOM tree.

  • document – the root of the DOM
  • html – child of document
  • head and body – children of html
  • Elements, text, and attributes – nodes inside body

Why We Use the DOM

  • Change text and HTML content
  • Change CSS styles
  • Respond to user events (clicks, typing, etc.)
  • Create interactive web pages

Selecting DOM Elements

Common ways to select elements:

  • document.getElementById()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
  • document.querySelector()
  • document.querySelectorAll()

What is Internal JavaScript?

Internal JavaScript is written inside the HTML file using the <script> tag.

It is usually placed at the end of the body so the HTML loads before JavaScript runs.
